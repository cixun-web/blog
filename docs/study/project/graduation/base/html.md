# HTMl

## 编写一个hello world页面

1. 创建文件
2. 编写文件
   ```html
   <!doctype html>
   <html lang="en">
     <head>
       <meta charset="UTF-8" />
       <meta
         name="viewport"
         content="width=device-width, user-scalable=no,  initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0"
       />
       <meta http-equiv="X-UA-Compatible" content="ie=edge" />
       <title>Document</title>
     </head>
     <body>
       hello word!
     </body>
   </html>
   ```
3. 运行文件

## 标签

### 常见标签

- p 段落

```html
<p>这是一个段落</p>
```

- a 超链接

```html
<a href="www.baidu.com">百度一下</a>
```

- img 图片

```html
<img src="***.jpg" alt="这是一张图片" />
```

- form 表单

```html
<form action=""></form>
```

- div 盒子

```html
<div>这是一个段落</div>
```
